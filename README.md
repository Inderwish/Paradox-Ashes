## 🌌 项目简介 (Introduction)

《悖论的灰烬》是一个**纯前端的互动式视听艺术作品**。

这是一个关于“极速空转”的文明故事。在这个单页应用中，我试图构建一种沉浸式的末世氛围——不仅仅通过文字，更通过**实时生成的音频（Procedural Audio）**和**动态的视觉粒子**来传达。

这并非传统的网页，而是一个运行在浏览器里的“数字墓碑”。

## ✨ 核心特性 (Features)

### 🎵 沉浸式音频引擎 (Audio Remaster)
摒弃了预录制的 MP3 文件，本作使用了 **Web Audio API** 实时合成音效：
* **动态嗡鸣 (Ambient Drone):** 由三个振荡器实时混合生成的背景音墙，带有模拟的呼吸感滤波器。
* **程序化音效:** 无论是点击终端的打字声，还是列车飞驰的撞击声，全部由代码实时演算生成。

### 👁️ 视觉特效系统
* **CRT 风格滤镜:** 模拟旧时代显示器的扫描线与辉光，营造复古科幻感。
* **粒子生态:** * `Beam`: 纯白的审判光束
    * `City`: 混合了骨灰的肮脏降雪
    * `Train`: 模拟超高速列车的视觉暂留线条
* **无缝主题切换:** 随着章节推进，页面的配色、光影与粒子逻辑会实时平滑过渡。

### 💻 极简主义技术栈
* **Zero-Dependency:** 没有复杂的打包工具，没有 `npm install`。
* **架构:** 原生 HTML5 / JavaScript (ES6+)。
* **样式:** Tailwind CSS (通过 CDN 引入)，实现了完全的响应式设计，手机也能完美体验。

## 🚀 如何运行 (How to Run)

1. **下载代码:** 点击右上角的 `Code` -> `Download ZIP`。
2. **解压:** 得到文件夹。
3. **运行:** 双击 `index.html`，直接在浏览器中打开。
4. **享受:** 记得**戴上耳机**，点击屏幕开启音频系统。
